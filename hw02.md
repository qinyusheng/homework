## 概念解释
### Information
```
任意形式存在的，与数据和知识相关，提供一些问题答案的物品
```
### Positional notation
```
是一种表示和编码数字的方法
```

### Algorithm
```
一种应用在数学上与计算机中的抽象的解决一类问题的方法
```

### Software Bug
```
一种出现在程序中，导致程序得出错误结果或者出现错误行为的错误
```
