1. AND gate
    - Boolean Expression : X = A * B
    - Logic Diagram Symbol : 
    
    ![](images/and.jpg)
    - Truth Table : 

        | A | B | X |
        |:--|:--|:--|
        | 0 | 0 | 0 |
        | 1 | 0 | 0 |
        | 0 | 1 | 0 |
        | 1 | 1 | 1 |
    - AND gate 是一种简单的逻辑电路，其作用是在输入两个电压时，能对两电压进行判断。当两电压都为高电压时，输出一个高电压。若两电压中有一个是低电压，则返回低电压。用 1 代表高电压，0 代表低电压时可表示为，输入 1 和 1 ，返回 1 。而输入的两个书中有一个是 0 时，返回 0 。
